# Windows BAT Utilities

A collection of console scripts for automating routine tasks in Windows. Each file solves a specific problem ‚Äî from system administration to gaming optimization.

## üìã Script List

### üîç CheckIp.bat  
Quick network port availability check.  
Usage: Run the file, enter IP address and port number.  
What it does:  
- Checks if the specified port is open
- Returns "OPEN" or "CLOSED" status
- Uses built-in Windows tools

### üßπ ClSteamCache.bat  
Cleans Steam client cache without deleting games.  
Usage: Run as administrator.  
What it cleans:  
- Download cache
- Update temporary files
- Local cached data
- Safe: does not affect installed games or saves

### üìÇ DelEmptyFolders.bat  
Recursive removal of all empty folders.  
Usage: Specify folder path when running.  
Features:  
- Works with any nesting level
- Ignores hidden system directories

### üéØ GameNumber.bat  
Simple console game "Guess the Number".  
Usage: Just run and follow the prompts.  
Features:  
- Random number generation from 1 to 10
- Replay option

### üõ°Ô∏è Scannow.bat  
File system integrity check.  
Usage: Run as administrator.  
What it checks:  
- Windows system files (via sfc /scannow)
- Recommended to run when disk issues are suspected
